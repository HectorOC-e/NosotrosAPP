# Cita ↔ Salida Link — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let a couple "start" a date idea so it becomes the active outing (budget) in Gastos, and surface "which date they're on" on Inicio.

**Architecture:** Add a nullable `budgets.date_idea_id` FK linking an outing to the date idea it came from. A `startDate(ideaId)` Server Action creates a new active budget named after the idea (short, editable). Inicio reads the active budget and, when it's linked to an idea, renders it as "CITA ACTUAL" with the full idea text.

**Tech Stack:** Next.js 15 App Router · TypeScript · Supabase (Postgres + RLS) · pnpm 11.1.1.

- [ ] **Step 1: Add `shortDateName` to `src/lib/format.ts`**

/** Short, editable outing name derived from a long date-idea sentence (≤32 chars). */
export function shortDateName(text: string): string {
  const t = text.trim().replace(/[.…]+$/, "");
  if (t.length <= 32) return t;
  const cut = t.slice(0, 32);
  const lastSpace = cut.lastIndexOf(" ");
  const base = (lastSpace > 0 ? cut.slice(0, lastSpace) : cut).replace(/[.,;:]+$/, "");
  return `${base}…`;
}
```

- [ ] **Step 2: Add `startDate` to `src/lib/actions/citas.ts`**

Add the import at the top (next to the existing imports):

```
Append the action:
```ts
/**
 * Starts a date: creates a new active outing (budget) named after the idea and
 * linked to it. The previous outing (if any) stays as history with its expenses.
 */
export async function startDate(dateIdeaId: string): Promise<{ ok: boolean }> {
  const { supabase, coupleId } = await requireCouple();

  const { data: idea } = await supabase
    .from("date_ideas")
    .select("text")
    .eq("id", dateIdeaId)
    .maybeSingle();
  if (!idea) return { ok: false };

  const { error } = await supabase.from("budgets").insert({
    couple_id: coupleId,
    label: shortDateName(idea.text),
    limit_amount: 0,
    date_idea_id: dateIdeaId,
  });
  if (error) throw error;

  revalidatePath("/citas");
  revalidatePath("/gastos");
  revalidatePath("/inicio");
  return { ok: true };
}
```
(`revalidatePath` and `requireCouple` are already imported in this file.)
